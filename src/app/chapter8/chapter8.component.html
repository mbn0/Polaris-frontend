ECB: Independent Block Encryption

In ECB mode, each plaintext block Pi is encrypted independently:

Ci = EK(Pi)

This simplicity allows full parallelism but leaks patterns when identical plaintext blocks recur.

Security Warning: Never use ECB for images or structured dataâ€”identical blocks produce identical ciphertext, revealing structure.
