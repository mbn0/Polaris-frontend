How RSA Works

1. Key Generation

Choose two large primes p and q, compute n = p×q and φ(n) = (p-1)(q-1).

2. Public Key

Select e such that gcd(e, φ(n)) = 1. Public key is (e, n).

3. Private Key

Compute d = e⁻¹ mod φ(n). Private key is (d, n).

4. Encrypt/Decrypt

Encrypt: c = m^e mod n. Decrypt: m = c^d mod n.
